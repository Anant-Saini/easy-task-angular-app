A beginner-friendly Angular task management project built to practice core frontend concepts such as component architecture, data-driven rendering, reusable UI design, and modern Angular development workflow.[page:1]

This repository shows hands-on progress in Angular development through iterative commits, including creating task-related features, experimenting with signal-based rendering, and refactoring the project structure from standalone components to module-based components.[page:1]

## How to Run Locally

### Prerequisites

- Node.js installed
- Angular CLI available in your environment
- npm for package management

### Installation

```bash
git clone https://github.com/Anant-Saini/easy-task-angular-app.git
cd easy-task-angular-app
npm install
```

### Start development server

```bash
ng serve
```

According to the repository README, the app runs locally at `http://localhost:4200/` during development.[page:1]

## Build

```bash
ng build
```

The repository README notes that the build output is generated in the `dist/` directory, following Angular’s default build behavior.[page:1]

## Testing

```bash
ng test
```

The current repository README mentions unit testing through Vitest via the Angular test command.[page:1]
